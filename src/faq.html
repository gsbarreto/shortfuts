shortfuts FAQ

Why isn't this extension working?

Your web app language must be in English. Other languages are currently not supported. If you're interested in helping me add support for another language, please contact me! If your web app language is English and it's still not working, please try force quitting Chrome and restarting.

Will this extension get me banned?

This extension was built with EA's ToS in mind. This extension only exposes keyboard shortcuts for things you can already accomplish in the web app, and does not modify the web app in any way. It's not a bot. It's not an auto-anything. If you use it responsibly, you shouldn't have any issues. That being said, *use at your own risk*. If you email me saying your account was banned or you were coin wiped, I will not respond.

Do I have to use Ctrl, Shift, or Alt with my hotkeys?

No you don't! Just open the extension popup and click the toggle. This will enabled "Legacy" mode where all the shortcuts are just 1 key. The only downside to using this mode is that keyboard shortcuts are not customizable. If you wish to have custom shortcuts, you have to use the native Chrome shortcut mode which forces you to use a modifier key.
